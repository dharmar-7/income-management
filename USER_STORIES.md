# User Stories — Income Management App

These describe what the app must do, written from the user's point of view.
Format: "As a [user], I want to [action], so that [reason]."

---

## Authentication

- As a user, I want to sign in with my Google account, so that I don't need a separate password and my data is secure.
- As a user, I want my session to stay active across visits, so that I don't have to log in every time.
- As a user, I want to log out from any device, so that I can protect my account if my device is lost.

---

## Google Pay Data Import

- As a user, I want to upload my Google Takeout JSON file, so that I can see my full transaction history from day one.
- As a user, I want to connect my Gmail account, so that new Google Pay transactions are automatically synced every hour.
- As a user, I want to manually upload a CSV or PDF, so that I can import transactions from other sources as a backup.
- As a user, I want to see a progress bar during import, so that I know the upload is working and how far along it is.
- As a user, I want duplicate transactions to be detected automatically, so that the same transaction isn't counted twice.

---

## Transactions

- As a user, I want to see all my transactions in a list, so that I can review my spending history.
- As a user, I want to filter transactions by date, category, and amount, so that I can find specific transactions quickly.
- As a user, I want each transaction to be auto-categorized by merchant (e.g., Swiggy → Food), so that I don't have to tag everything manually.
- As a user, I want to edit a transaction's category or add a note, so that I can correct mistakes or add context.

---

## Budgets & Goals

- As a user, I want to set a monthly budget for each spending category, so that I know my limits for the month.
- As a user, I want to see a progress bar for each budget, so that I can see at a glance how much I've spent vs. my limit.
- As a user, I want to receive an alert when I've used 80% of a budget, so that I can slow down spending before I go over.
- As a user, I want to set a savings goal with a target amount, so that I can track how much I'm saving each month.

---

## Analytics & Reports

- As a user, I want to see charts showing my income vs. expenses over time, so that I can understand my financial trends.
- As a user, I want to see a breakdown of spending by category, so that I know where most of my money goes.
- As a user, I want to download a monthly PDF report, so that I can keep a record or share it with my accountant.
- As a user, I want to export my transactions as a CSV file, so that I can analyse them in Excel or Google Sheets.

---

## Mobile App

- As a user, I want to use the app on my phone, so that I can check my finances anywhere.
- As a user, I want to unlock the app with my fingerprint, so that my financial data is protected even if someone picks up my phone.
- As a user, I want to see my recent transactions offline, so that I can access my data without an internet connection.

---

## Notes

- As a user, I want to create notes with a title and markdown content, so that I can capture ideas, checklists, and reminders alongside my finances.
- As a user, I want to assign a color to each note, so that I can visually organise them at a glance.
- As a user, I want a "Mirror" glass effect color option, so that my notes look modern and visually distinct.
- As a user, I want to pin important notes to the top, so that they are always easy to find.
- As a user, I want to archive notes I no longer need, so that my main view stays clean without losing data.
- As a user, I want to add tags to notes and filter by tag, so that I can group related notes quickly.
- As a user, I want to set a reminder on a note, so that I get notified at the right time.
- As a user, I want to attach images to a note, so that I can store receipts or photos alongside my text.
- As a user, I want to lock a note with a password, so that sensitive notes are protected even if someone opens the app.
- As a user, I want to search my notes by keyword, so that I can find what I need without scrolling.

---

## Security

- As a user, I want my data to be encrypted, so that even if there is a breach, my information is unreadable.
- As a user, I want to be able to disconnect my Gmail account, so that I can revoke access at any time.
- As a user, I want to know my Google Pay tokens are stored securely, so that my account can't be accessed by others.
